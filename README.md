## Project Overview <br>
This project implements a multiple linear regression model to quantify the impact of three advertising channels - TV, Radio, and Newspaper - on weekly sales. The analysis is anchored in the theory and practical methods presented Applications in Python (James et al., 2023).
<br> The main goal is to accurately measure channel effectiveness, validate the model, and provide evidence-based recommendations to optimize marketing spend.
